\# 📘 README — Computability \& Complexity: Preliminaries Study Guide



> Based on the lecture slides by \*\*Ugo Dal Lago\*\*

\## 📚 Subjects Covered



\### Section 1 — History of Computation

\- Origins of computation (pre-computer era)

\- Euclid's Algorithm as an early example of computation

\- Evolution from "art" to scientific discipline

\- The Modern Theory of Computation (ToC)

\- Impact of ToC on other sciences and ICT



\### Section 2 — Modelling Computation

\- The Standard Paradigm: Tasks vs. Processes

\- \*\*The Multiplication Problem\*\*

&nbsp; - RepeatedAddition process

&nbsp; - GridMethod (school algorithm) process

&nbsp; - Comparing efficiency between processes

\- \*\*The Wedding Problem\*\*

&nbsp; - Exhaustive enumeration and its limits

&nbsp; - Introduction to the classes \*\*P\*\* and \*\*NP\*\*

&nbsp; - The P vs. NP open question

\- What is an Algorithm? (3 core constraints)

\- Why proving non-existence of algorithms is hard

\- Open Research Questions in Complexity Theory



\### Section 3 — Mathematical Preliminaries

\- \*\*Sets and Numbers\*\*

&nbsp; - Cardinality, natural numbers (ℕ), integers (ℤ)

&nbsp; - Ceiling notation ⌈x⌉

&nbsp; - Set notation \[n]

&nbsp; - Logarithm base 2 convention



\- \*\*Strings\*\*

&nbsp; - Alphabets and string definition

&nbsp; - String sets: Sⁿ and S\*

&nbsp; - Empty string ε

&nbsp; - Concatenation and repetition

&nbsp; - String length notation



\- \*\*Functions\*\*

&nbsp; - Cartesian products

&nbsp; - Relations

&nbsp; - Functions as a special type of relation



\- \*\*Tasks as Functions\*\*

&nbsp; - Encoding tasks as functions from {0,1}\* to {0,1}\*

&nbsp; - Encoding objects (text, numbers, pairs) as binary strings



\- \*\*Languages and Decision Problems\*\*

&nbsp; - Boolean / characteristic functions

&nbsp; - Language definition

&nbsp; - Decision problems as language membership



\- \*\*Asymptotic Notation\*\*

&nbsp; - Big-O — upper bound (at most)

&nbsp; - Big-Omega (Ω) — lower bound (at least)

&nbsp; - Big-Theta (Θ) — tight bound (exactly)

&nbsp; - Using limits to determine asymptotic relationships
